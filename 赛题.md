# 电价预测与储能优化赛题说明

## 1. 赛题介绍

随着新能源装机规模的持续增长，电力系统运行的不确定性不断增加，电力市场价格波动也愈发显著。

储能系统由于具有灵活的充放电能力，可以在电价较低时充电，在电价较高时放电，从而实现电能在时间维度上的转移，并获得电力市场收益。

在电力现货市场中，储能系统的收益通常来源于电价时间波动所带来的时序调节价值。如果能够提前对未来电价进行准确预测，并在储能容量与充放电约束条件下制定合理的运行策略，则可以有效提升储能系统的经济运行水平。

本赛题基于蒙西地区某节点的历史实时电价数据及相关边界条件，要求参赛队伍：

1. 构建模型预测未来一天，即 **D+1 日** 的实时市场电价；
2. 根据预测电价制定储能设备的充放电计划；
3. 在给定约束条件下，使储能系统获得最大化收益。

---

## 2. 背景知识

### 2.1 储能系统基本运行策略

储能系统在电力市场中的基本运行策略是：

> **低价充电，高价放电。**

设实时电价序列为：

$$
P_t
$$

其中：

| 符号    | 含义                                                      |
| ----- | ------------------------------------------------------- |
| $t$   | 时间索引。本赛题采用 **15 分钟分辨率**，未来 96 个时间点，即 $0 \leq t \leq 95$ |
| $P_t$ | 时间 $t$ 的实时电价                                            |

储能系统在时间 $t$ 的充放电功率为：

$$
E_t
$$

其中：

| 取值            | 含义    |
| ------------- | ----- |
| $E_t = +1000$ | 表示放电  |
| $E_t = -1000$ | 表示充电  |
| $E_t = 0$     | 表示不操作 |

储能系统的收益可表示为：

$$
Profit = \sum_{t=0}^{95} P_t \cdot E_t
$$

---

### 2.2 电价预测难点

电价序列通常具有以下特征：

* **非平稳性**：电价分布和波动规律会随时间变化；
* **高波动性**：受供需、天气、市场行为等因素影响，电价可能出现剧烈变化；
* **多因素耦合**：电价同时受到负荷、新能源出力、气象条件、联络线功率和市场行为等因素影响。

因此，实时电价预测具有较高难度。

常见建模技术包括：

* 深度时间序列模型：如 **LSTM、Transformer**；
* 时序卷积网络：如 **TCN**；
* 信号分解方法：如 **VMD、EMD**；
* 大模型方法；
* 其他机器学习或集成学习方法。

参赛选手可根据自身思路选择适合的技术路线进行建模。

## 3. 比赛任务

本赛题要求参赛队伍基于以下数据，对 **蒙西地区某节点的实时市场电价** 进行建模与预测：

* 历史价格数据；
* 边界条件数据；
* 气象预报信息。

其中边界条件包括：

* 系统负荷；
* 风光总加；
* 竞价空间；
* 联络线；
* 风电；
* 光伏；
* 水电；
* 非市场化机组等相关数据。

### 3.1 预测目标

预测区间为：

> **次日 0 时起未来 24 小时的实时电价序列。**

数据分辨率为：

> **15 分钟分辨率，共 96 个点。**

### 3.2 策略优化目标

在获得未来电价预测结果的基础上，参赛队伍需要在给定储能运行约束下制定储能设备的充放电计划，从而实现每日收益最大化。

## 4.1 气象数据：`all_nc` 目录

气象数据来源为行业主流气象预报，即 **Numerical Weather Prediction，NWP**。

文件格式为：

```text
.nc
```

每个文件共包含 **7 个变量**。

### 4.1.1 文件含义

每个文件为发布日对应的未来 1 天逐小时预报。

例如：

```text
20240101.nc
```

表示：

> 2024 年 1 月 1 日发布、对 2024 年 1 月 2 日的逐小时气象预测。

### 4.1.2 文件维度

文件维度包括：

| 维度        | 含义                  |
| --------- | ------------------- |
| `time`    | 第一个预报时刻，世界时         |
| `channel` | 变量维度，共 7 个变量        |
| `hour`    | 从起始时间起的小时数，范围为 0–23 |
| `lat`     | 纬度，由北向南递增           |
| `lon`     | 经度，由西向东递增           |

参赛队伍可自主选择使用部分或全部变量。

本赛题气象数据集由 **中科天机气象科技有限公司（tjweather.com）** 提供。

## 4.2 边界条件数据

边界条件数据为 **15 分钟分辨率**，时间为 **北京时间**。

### 4.2.1 系统负荷

| 特征      | 说明                                                       |
| ------- | -------------------------------------------------------- |
| 系统负荷实际值 | 反映电力系统在各时间点的真实用电需求水平，是影响电力供需关系及市场出清电价的重要基础指标。            |
| 系统负荷预测值 | 由调度或市场机构对未来时段系统用电需求进行预测得到，用于提前评估电力供需平衡情况，并影响市场参与主体的交易决策。 |

### 4.2.2 风光总加

| 特征      | 说明                                                      |
| ------- | ------------------------------------------------------- |
| 风光总加实际值 | 表示风电与光伏发电出力之和的实际值，反映新能源在系统中的实际发电水平，对市场供给侧结构及电价波动具有重要影响。 |
| 风光总加预测值 | 根据气象条件等因素对未来风电与光伏总出力进行预测得到，用于评估未来新能源供给水平及其对电力市场价格的潜在影响。 |

### 4.2.3 联络线

| 特征     | 说明                                                   |
| ------ | ---------------------------------------------------- |
| 联络线实际值 | 表示跨区域电网联络线的实际受送电功率，反映区域之间电力交换情况，对本地区供需平衡和市场价格具有调节作用。 |
| 联络线预测值 | 对未来时段跨区域电网联络线的受送电功率进行预测得到，用于评估区域电力交换对本地电力供需格局的影响。    |

### 4.2.4 风电

| 特征    | 说明                                     |
| ----- | -------------------------------------- |
| 风电实际值 | 表示风电机组在各时间点的实际发电出力，反映风资源条件下风电的真实发电水平。  |
| 风电预测值 | 根据气象预报对未来风电发电出力进行预测得到，是评估新能源供给能力的重要依据。 |

### 4.2.5 光伏

| 特征    | 说明                                                |
| ----- | ------------------------------------------------- |
| 光伏实际值 | 表示光伏电站在各时间点的实际发电出力，反映太阳辐照条件下的真实发电水平。              |
| 光伏预测值 | 根据气象预报对未来光伏发电出力进行预测得到，用于评估未来新能源发电水平及其对电力市场供给侧的影响。 |

### 4.2.6 水电

| 特征    | 说明                                        |
| ----- | ----------------------------------------- |
| 水电实际值 | 表示水电机组在各时间点的实际发电出力，通常受来水情况及调度计划影响。        |
| 水电预测值 | 根据水情及调度计划对未来水电出力进行预测得到，用于评估水电资源在系统供给中的贡献。 |

### 4.2.7 非市场化机组

| 特征        | 说明                                             |
| --------- | ---------------------------------------------- |
| 非市场化机组实际值 | 表示因保供、电网安全或政策原因不参与市场竞价的机组实际发电出力，这部分电量通常优先保障运行。 |
| 非市场化机组预测值 | 对未来时段非市场化机组的计划或预测出力进行估计，用于评估固定出力对市场供需结构的影响。    |

## 6. 储能参数设定

为统一比赛条件，本赛题对储能系统进行简化建模，储能设备具有固定容量与离散充放电策略。

储能系统参数设定如下：

| 参数          | 数值 / 规则            |
| ----------- | ------------------ |
| 储能容量        | 8000               |
| 充放电功率       | ±1000              |
| 单次充电或放电持续时间 | 连续 8 个时间点，即连续 2 小时 |

## 7. 储能运行规则

### 7.1 容量约束

储能能量状态需要满足：

$$
0 \leq SOC_t \leq 8000
$$

---

### 7.2 充放电规则

储能系统仅允许以下两种完整操作。

#### 7.2.1 充电操作

```text
[-1000, -1000, -1000, -1000, -1000, -1000, -1000, -1000]
```

#### 7.2.2 放电操作

```text
[+1000, +1000, +1000, +1000, +1000, +1000, +1000, +1000]
```

也就是说：

> 每次操作必须持续 **连续 8 个时间点**，不可中断。

---

### 7.3 每日操作限制

在一天 24 小时内，最多允许一次完整充放电操作。

即允许以下两种情况：

1. 不进行交易；
2. 进行一次“充电 + 放电”操作。

若选择充电，则必须在当天完成对应的放电操作。

---

### 7.4 时间约束

充电开始时间满足：

$$
0 \leq t_c \leq 80
$$

放电开始时间满足：

$$
t_d \geq t_c + 8
$$

且：

$$
t_d \leq 88
$$

其中：

| 符号    | 含义     |
| ----- | ------ |
| $t_c$ | 充电开始时间 |
| $t_d$ | 放电开始时间 |

---

## 8. 评测指标

评测时，对测试集对应天数计算储能系统收益。

### 8.1 单日收益

收益计算公式为：

$$
Profit = \sum_{t=0}^{95} P_t \cdot E_t
$$

其中：

| 符号    | 含义           |
| ----- | ------------ |
| $P_t$ | 真实实时电价       |
| $E_t$ | 参赛队伍提交的充放电功率 |

若某天未进行充放电操作，则该日收益记为：

$$
Profit = 0
$$

---

### 8.2 最终评分

最终评分为测试集所有天数收益的平均值：

$$
Score = \frac{1}{D} \sum_{d=1}^{D} Profit_d
$$

其中：

| 符号         | 含义         |
| ---------- | ---------- |
| $D$        | 测试集天数      |
| $Profit_d$ | 第 $d$ 天的收益 |

最终目标为：

> **收益越高，排名越靠前。**

## 11. 赛题核心总结

本赛题本质上包含两个连续任务：

### 11.1 电价预测任务

利用历史实时电价、边界条件和气象预报信息，预测次日 96 个时间点的实时市场电价。

### 11.2 储能优化任务

基于预测电价，在储能容量、充放电功率、连续操作时长和每日操作次数限制下，选择最优的充电时间段和放电时间段，使真实收益最大化。

简化理解为：

> 先预测明天每 15 分钟的电价，再找出一天中最适合连续 2 小时充电和连续 2 小时放电的时间段，最终让“低价买入、高价卖出”的收益最大。
